# Jules Bake-Off Workspace

This directory is the owned bake-off workspace/output root for Jules.

Use it for:

- deliverables
- logs
- patches
- screenshots

Subdirectories:

- `deliverables\`
- `logs\`
- `patches\`
- `screenshots\`

This folder is intended to keep Jules-produced artifacts separate from Antigravity-produced artifacts during the bake-off.

Actual code edits should occur in the isolated Jules repo lane. This folder is the canonical drop zone for exported patches, logs, screenshots, and written submissions.
