# Antigravity Bake-Off Workspace

This directory is the owned bake-off workspace/output root for Antigravity.

Use it for:

- deliverables
- logs
- patches
- screenshots

Subdirectories:

- `deliverables\`
- `logs\`
- `patches\`
- `screenshots\`

This folder is intended to keep Antigravity-produced artifacts separate from Jules-produced artifacts during the bake-off.

Actual code edits should occur in the isolated Antigravity repo lane. This folder is the canonical drop zone for exported patches, logs, screenshots, and written submissions.
